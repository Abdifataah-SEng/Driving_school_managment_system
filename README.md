# Driving School Management System

A simple Windows Forms application written in C# targeting `.NET Framework 4.7.2` that helps manage common driving school tasks. The app provides user authentication and a basic dashboard for managing students, instructors, lesson schedules, payments, and tracking student progress.

## Key features

- User login and password recovery
- Student and instructor management (create, edit, remove)
- Lesson scheduling and attendance tracking
- Payment recording and basic fee management
- Progress tracking for student lessons
- Basic dashboard for quick overview and reporting

## Tech stack

- C# and Windows Forms
- `.NET Framework 4.7.2`
- Built with Visual Studio

## Getting started

1. Open the solution in Visual Studio.
2. Make sure `.NET Framework 4.7.2` is installed on your machine.
3. Build the solution and run the startup project (usually the main Windows Forms project).
4. If the app requires a database, configure the connection string in `App.config` or the appropriate settings file before running.
